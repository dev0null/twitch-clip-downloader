## Name
Twitch Clip Downloader

## Description
Twitch Clip Downloader is a CLI tool for downloading clips from "twitch.tv"

## Usage

Make it executable:

``` 
chmod +x twitchclip
```

Run it providing a clip url as a parameter:

```
./twitchclip "https://url.here"
```

## Requirements

- **wget**

- **Brave Browser** or **Google Chrome**
